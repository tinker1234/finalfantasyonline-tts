# finalfantasyonline-tts
voice dialogue only works in fullscreen
source and binary (https://drive.google.com/file/d/1itcBoIewbjSC7Tp0KDUPX8a6DXFRcJX_/view?usp=sharing)
 unless I'm mistaken (and don't quote me), you should be able to use this without getting banned because it doesn't modify the client in any way. It doesn't add an advantage. It just takes screenshots and reads them with OCR. It's faster on Windows. Haven't tested it on Linux or Mac. 
The newest GPU should be supported with the EXE. I'm not sure if the requirements installed the right CUDA.
